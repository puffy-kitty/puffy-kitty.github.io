# puffy-kitty.github.io

Crochet pattern sharing site for Puffy Kitty.

The site displays each crochet pattern on its own detail page, supports Chinese/US/UK crochet terms, and can generate downloadable PNG and PDF files in the browser.

## Files

- `index.html`: website content and pattern cards
- `styles.css`: page styling
- `scripts.js`: browser-side PNG/PDF generation
- `assets/`: visual references and pattern preview images
- `patterns/`: optional folder for future images or downloadable assets

## GitHub Pages

This repository is named `puffy-kitty.github.io`, so the site URL is:

```text
https://puffy-kitty.github.io/
```

In GitHub, open `Settings > Pages` and use:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

## Add A New Pattern

1. Add a new entry in `patternData` inside `scripts.js`.
2. Provide `zh`, `us`, and `uk` text for the title, summary, rows, notes, hair, and abbreviations.
3. The homepage card and detail page are rendered automatically from the pattern data.
4. Download buttons use the current language selected by the visitor.

```html
downloadPatternImage(pattern, selectedLanguage);
downloadPatternPdf(pattern, selectedLanguage);
```

The download files are generated by the visitor's browser, so pre-made PDF files are not required.

## Visual Direction

The current visual direction follows the older image-format pattern sheets:

- pale blue background
- rounded, cute, readable black lettering
- hand-notebook style highlights
- photo previews with soft, irregular rounded corners
- small doodle accents instead of commercial product cards
