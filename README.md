# puffy-kitty.github.io

Crochet pattern sharing site for Puffy Kitty.

The site uses a layered artist-archive structure: a personal homepage, series pages, gallery pages, and individual pattern reader pages. Pattern readers support Chinese/US/UK crochet terms and can generate downloadable PNG and PDF files in the browser.

## Files

- `index.html`: website content and pattern cards
- `styles.css`: page styling
- `scripts.js`: browser-side routing, pattern Markdown loading, and PNG/PDF generation
- `assets/`: visual references and pattern preview images
- `patterns.json`: ordered list of pattern folders to load
- `patterns/gummy/pokemon/ralts/`: current pattern folder with Markdown content and images

## Font

The site uses `cjkFonts 全瀨體` through ZeoSeven Fonts:

```css
@import url("https://fontsapi.zeoseven.com/200/main/result.css");
```

ZeoSeven lists the font under OFL-1.1. The current implementation uses the hosted WebFont instead of committing the 30MB font file into this repository.

## GitHub Pages

This repository is named `puffy-kitty.github.io`, so the site URL is:

```text
https://puffy-kitty.github.io/
```

In GitHub, open `Settings > Pages` and use:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

## Local Preview

Because the site loads `patterns.json` and each pattern's `pattern.MD` in the browser, it must be opened through `http://`, not by double-clicking `index.html`.

```powershell
node serve-local.js
```

Then open:

```text
http://127.0.0.1:4173/
```

## Add A New Pattern

1. Create a new folder under `patterns/` using layered archive paths, for example `patterns/gummy/pokemon/ralts/`.
2. Put the pattern text in `pattern.MD`.
3. Put the cover image and detail images in the same folder, for example `cover.JPEG`, `1.JPEG`, `2.JPEG`, `3.JPEG`.
4. Add the folder path to `patterns.json`.
5. The pattern series page and detail page are rendered automatically from the Markdown file.
6. Download buttons use the current language selected by the visitor.

```text
patterns/gummy/pokemon/ralts/
  pattern.MD
  cover.JPEG
  1.JPEG
  2.JPEG
  3.JPEG
```

The download files are generated by the visitor's browser, so pre-made PDF files are not required.

## Page Structure

- `#/`: personal homepage with hero image, introduction, and series rows
- `#/pattern`: crochet pattern series gallery
- `#/pattern/patterns/gummy/pokemon/ralts`: individual pattern reader page
- `#/gallery`: OC/custom gallery page
- `#/about`: site workflow notes
- `#/contact`: contact page

## Visual Direction

The current visual direction follows the older image-format pattern sheets:

- pale blue background
- rounded, cute, readable black lettering
- hand-notebook style highlights
- photo previews with soft, irregular rounded corners
- small doodle accents instead of commercial product cards
