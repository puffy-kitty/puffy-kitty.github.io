# puffy-kitty.github.io

Crochet pattern sharing site for Puffy Kitty.

The site displays each crochet pattern directly on the page and can generate downloadable PNG and PDF files in the browser.

## Files

- `index.html`: website content and pattern cards
- `styles.css`: page styling
- `scripts.js`: browser-side PNG/PDF generation
- `patterns/`: optional folder for future images or downloadable assets

## GitHub Pages

This repository is named `puffy-kitty.github.io`, so the site URL is:

```text
https://puffy-kitty.github.io/
```

In GitHub, open `Settings > Pages` and use:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

## Add A New Pattern

1. Copy the existing `<article class="pattern-card">...</article>` block in `index.html`.
2. Change the pattern title, description, and displayed instructions.
3. Add a matching entry in `patternData` inside `scripts.js`.
4. Point the buttons to the new pattern id:

```html
<button type="button" data-download-image="new-pattern-id">下载图片</button>
<button type="button" data-download-pdf="new-pattern-id">下载 PDF</button>
```

The download files are generated by the visitor's browser, so pre-made PDF files are not required.
