## abbreviations

### zh

- x = 短针，v = 加针，A = 减针，B = 泡泡针，ch = 锁针，SL = 引拔针，T = 中长针，F = 长针。

### us

- sc = single crochet, inc = increase, dec = decrease, bobble = bobble stitch, ch = chain, SL st = slip stitch, hdc = half double crochet, dc = double crochet.

### uk

- dc = double crochet, inc = increase, dec = decrease, bobble = bobble stitch, ch = chain, SL st = slip stitch, htr = half treble crochet, tr = treble crochet.
